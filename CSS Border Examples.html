CSS Border Examples

A collection of different border styles and techniques

Different Colored Borders
Each side has a different color
/* Different colored borders for each side */ .different-colors { border: 6px solid; border-top-color: #e74c3c; /* Red */ border-right-color: #3498db; /* Blue */ border-bottom-color: #2ecc71; /* Green */ border-left-color: #f39c12; /* Orange */ }
Dotted, Dashed & Solid Borders
Different border styles
/* Different border styles for each side */ .border-styles { border: 6px solid; border-top-style: dotted; /* Dotted line */ border-right-style: dashed; /* Dashed line */ border-bottom-style: solid; /* Solid line */ border-left-style: double; /* Double line */ /* Different colors for each side */ }
Double Border
Double border effect
/* Double border style */ .double-border { border: 8px double #8e44ad; /* Double border with purple color */ }
Rounded Border
Border with rounded corners
/* Rounded border using border-radius */ .rounded-border { border: 5px solid #e74c3c; /* Solid red border */ border-radius: 25px; /* Round the corners */ }
Gradient Border
Border with gradient colors
/* Gradient border using background-clip technique */ .gradient-border { border: 4px solid transparent; /* Transparent border */ background: linear-gradient(white, white) padding-box, /* Solid color for inside */ linear-gradient(to right, #ff9a9e, #fad0c4, #a18cd1) border-box; /* Gradient for border */ }
Border with Shadow
Border with shadow effect
/* Border with shadow effect */ .shadow-border { border: 3px solid #3498db; /* Solid blue border */ box-shadow: 0 0 15px rgba(52, 152, 219, 0.8); /* Blue glow effect */ }
Curved Button Border
/* Curved button with border-radius */ .curved-button { padding: 12px 30px; background: linear-gradient(to right, #3498db, #2c3e50); color: white; border: none; border-radius: 30px; /* Creates the curved border effect */ box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); transition: all 0.3s ease; /* Smooth hover effect */ } .curved-button:hover { transform: translateY(-3px); /* Lift effect on hover */ box-shadow: 0 6px 12px rgba(0, 0, 0, 0.3); /* Enhanced shadow on hover */ }
